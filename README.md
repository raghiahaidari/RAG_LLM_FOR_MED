# RAG test with llama3 model

## Setup

Create a virtual environment and install the required packages :

For Linux :
```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

For windows :
```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
